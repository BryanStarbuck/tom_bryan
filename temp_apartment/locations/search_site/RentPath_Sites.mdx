## What it is and how it works

**Four brands, one database.** Rent.com, ForRent.com, ApartmentGuide, and ApartmentFinder are
all RentPath properties, now under **Redfin**. They share the same listing inventory behind
different skins.

**The practical consequence:** if you search Rent.com and then search ForRent.com, you have
searched the same database twice. Different logo, same results. **Searching more than one is
wasted effort** — pick whichever loads and move on.

**Useful number:** ~**334 furnished Redmond apartments**. Furnished matters here — the
[overview](../../overview.mdx) found that unfurnished is a trap for a 2-month stay once
[CORT furniture rental](../corporate_housing/CORT_Furniture_Rental.mdx), delivery, two moves,
and utility setup are added.

## How to search it effectively

Furnished Redmond (Rent.com):

```
https://www.rent.com/washington/redmond-apartments/furnished
```

Short-term Bellevue (ForRent):

```
https://www.forrent.com/find/WA/metro-Seattle/Bellevue/extras-Short+Term+Lease
```

**Two practical warnings:**

1. **Rent.com rate-limits aggressively.** Expect to be throttled or blocked if you page
   through results quickly. Slow down or switch to a sibling brand — same data.
2. **Do not cross-check between the brands.** It is the same database. Cross-checking creates
   false confidence and duplicate call lists.

## Does the filter actually work?

**The filter exists. The result counts are inflated — badly.**

**ForRent claims 977 Bellevue short-term results.** That number is not real inventory. Here is
the mechanism:

> **RentPath counts BUILDINGS that offer short terms and multiplies by floor plans.**

A single building with 12 floor plans that once wrote a short lease contributes **12 results**
to the count. Roughly 80 such buildings produce "977 short-term results" without a single
30-day unit existing anywhere. The number measures *floor plan brochures*, not available
apartments.

This is the same property-level flag problem found on [Apartments.com](./Apartments_com.mdx) —
"we sometimes do short terms" — but amplified by a multiplier that turns it into an
impressive-looking headline.
