## What it is and how it works

**PadMapper is Zumper-owned.** That is the fact that governs everything on this page. It
draws from the same inventory and has the same real, structured **Short Term toggle** — not
a keyword hack. It is **strictly a subset of Zumper** (20 results here vs. 25 there).

So why use it at all? **The map.** PadMapper's map UI is materially better than Zumper's
list, and geography is a live constraint in this search — the charter cares about Redmond
first, East Bellevue second, South Kirkland third, all inside a 20-minute drive. A map answers
"is this actually in East Bellevue or is it Factoria?" faster than any list can.

**The useful finding:** Redmond 1BRs on PadMapper **cluster $2,010–2,730** — comfortably
under the $3,300 ceiling, in the preferred city. That price band is the most encouraging
signal any mainstream site produced.

## How to search it effectively

```
https://www.padmapper.com/apartments/redmond-wa/short-term
https://www.padmapper.com/apartments/bellevue-wa/short-term
https://www.padmapper.com/apartments/kirkland-wa/short-term
```

Toggle: **Short Term**.

**The efficient division of labor:**

* **PadMapper → geography.** Pan the map across Redmond, Crossroads/Lake Hills, and South
  Kirkland. Confirm a listing is where the charter wants it before spending time on it.
* **[Zumper](./Zumper.mdx) → the list.** It returns the fuller set (25 vs 20) with the same
  filter.

**Do not run both exhaustively.** PadMapper is a subset — anything it shows, Zumper shows
too. The map is the only reason to open it.

## Does the filter actually work?

**Yes.** It inherits Zumper's real structured Short Term field, which makes these two sites
the only mainstream search surfaces in this category where the filter means what it says.
Everywhere else it is a keyword match ([Zillow](./Zillow.mdx)), a property-level flag
([Apartments.com](./Apartments_com.mdx)), marketing ([Apartment List](./Apartment_List.mdx)),
or absent entirely ([HotPads](./HotPads.mdx), [Trulia](./Trulia.mdx)).

**The caveats are Zumper's caveats:**

* Result counts look **capped**, not complete.
* Listings can be **syndicated duplicates** of [Apartments.com](./Apartments_com.mdx).
* "Short term" in this industry typically means **3–6 months**, not the 1–2 Tom needs. A
  working filter still cannot produce inventory that does not exist.

Treat results as a **call list**. The filter gets you a phone number worth dialing; it does
not get you a 30-day lease.
